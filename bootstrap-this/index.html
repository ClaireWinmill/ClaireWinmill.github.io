Colors

Colors can add meaning to text and elements, and can be added using a set of color utility classes. These classes are not named after the colors themselves (red, blue etc.)

These classes can change the color of the text itself

Borders

Use border utilities to quickly style the border and border-radius of an element. Can be used for images, buttons, and other elements

Bootstrap color classes can be added to .border

Additive or subtractive classes can be used to add or take away an elements borders

Buttons

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

The .btn class is designed to be used with the < button > tag. However, you can also use this class on < a > tags or < input > elements

Link

Bootstrap color classes can also be applied to buttons

You can add "outline" to your .btn class to remove the button color unless hovered over. Give it a try!

Adding a class of .btn-lg to the button tag will allow you to create a large button

Adding a class of .btn-sm to the button tag will allow you to create a small button

Adding a class of .btn-block to the button tag will allow you to create a button that spans the full width of its parent element
