Tighten up your Bootstraps!

On this page you will be introduced to Bootstrap, the world’s most popular framework for building responsive, mobile-first sites

With this system, you can prototype your ideas or build your entire app with Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Bootstrap can be used to style

  • Content
  • Utilities
  • Components

small tag for small or fine print.

strong tag for bold text.

em tag for italicized text.

Utilities


Colors

Colors can add meaning to text and elements, and can be added using a set of color utility classes. These classes are not named after the colors themselves (red, blue etc.)

These classes can change the color of the text itself

text-secondary

text-success

Alerts
